This files explains the organization of the folders and how to generate the compiled PDF.
------------------------------------------------------------------------------------------------------------------------------------------------
In the chapter folders there are all the .tex files where the different chapters are written.

In the images folder there are all the pictures required for the thesis organized in folders by chapters.

In the main folder there are all the files required to compile the code in LATEX.
------------------------------------------------------------------------------------------------------------------------------------------------
The compile file describes how the compilation should be done in Ubuntu machine language.
------------------------------------------------------------------------------------------------------------------------------------------------
Finally, the doc.pdf file in the root folder contains the final compiled document.
------------------------------------------------------------------------------------------------------------------------------------------------
